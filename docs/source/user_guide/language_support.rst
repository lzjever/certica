Language Support
================

Certica supports multiple languages in UI mode only.

Supported Languages
-------------------

- ``en`` - English (default)
- ``zh`` - Chinese (中文)
- ``fr`` - French (Français)
- ``ru`` - Russian (Русский)
- ``ja`` - Japanese (日本語)
- ``ko`` - Korean (한국어)

Usage
-----

.. code-block:: bash

   certica ui --lang zh

Important Notes
---------------

- The ``--lang`` option is only available in UI mode
- CLI commands always use English for script compatibility
- If an unsupported language is specified, the tool will warn and fall back to English
